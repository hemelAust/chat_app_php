$project
========

Simple chat application where you can chat with your friends.

Look how easy it is to use:
---------------------------

  Download the project.
  Extract the project.
  import the chat.sql file into the db.
  run any of your browser you feel comfort.

Features
--------

- Instant chat.
- Alert message for any errors.
- validated fields.
- Secure chating.

Installation
------------

Install $project by running:

    	Download the project.
  	Extract the project.
  	import the chat.sql file into the db.
  	run any of your browser you feel comfort
